# C - Pointers, Arrays and Strings

This project introduces fundamental concepts in C programming including pointers, arrays, and strings.

## Learning Objectives

- What are pointers and how to use them
- What are arrays and how to use them
- What are strings and how to manipulate them
- The differences between pointers and arrays
- How to use strings and how to manipulate them
- Scope of variables

## Requirements

- All files compiled on Ubuntu 20.04 LTS with gcc using flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows Betty style
- No global variables
- No more than 5 functions per file
- Only allowed to use `_putchar` (no printf, puts, etc.)
- Header file `main.h` must contain all function prototypes

## Files Description

| File Name | Description |
|-----------|-------------|
| `0-reset_to_98.c` | Takes a pointer to an int and updates the value it points to to 98 |
| `1-swap.c` | Swaps the values of two integers |
| `2-strlen.c` | Returns the length of a string |
| `3-puts.c` | Prints a string to stdout |
| `4-print_rev.c` | Prints a string in reverse |
| `5-rev_string.c` | Reverses a string |
| `6-puts2.c` | Prints every other character of a string |
| `7-puts_half.c` | Prints half of a string |
| `8-print_array.c` | Prints n elements of an array of integers |
| `9-strcpy.c` | Copies a string |
| `main.h` | Header file with all function prototypes |
| `_putchar.c` | Custom `_putchar` function (for local testing) |
