# C - Debugging

This project focuses on debugging techniques in C programming. Debugging is the process of finding and fixing errors in code.

## Learning Objectives

- What is debugging
- What are some methods of debugging manually
- How to read error messages
- How to use `printf` to debug
- How to use `gdb` to debug

## Requirements

- All files compiled on Ubuntu 20.04 LTS with gcc using flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows Betty style
- No global variables
- No more than 5 functions per file

## Files Description

| File Name | Description |
|-----------|-------------|
| `0-main.c` | Test file for `positive_or_negative()` function with input 0 |
| `main.h` | Header file containing function prototypes |

## Tasks
