# C - Functions, Nested Loops

This project covers functions and nested loops in C programming.

## Requirements
- All files compiled on Ubuntu 20.04 LTS with gcc using flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows Betty style
- No global variables
- No more than 5 functions per file

## Files Description

| File | Description |
|------|-------------|
| `0-putchar.c` | Prints `_putchar` followed by a new line |
| `_putchar.c` | Contains the `_putchar` function for printing characters |
| `1-alphabet.c` | Prints the alphabet in lowercase |
| `2-print_alphabet_x10.c` | Prints alphabet 10 times |
| `3-islower.c` | Checks for lowercase character |
| `4-isalpha.c` | Checks for alphabetic character |
| `5-sign.c` | Prints the sign of a number |
| `6-abs.c` | Computes absolute value |
| `7-print_last_digit.c` | Prints last digit of a number |
| `8-24_hours.c` | Prints every minute of the day |
| `9-times_table.c` | Prints 9 times table |
| `10-add.c` | Adds two integers |
| `11-print_to_98.c` | Prints numbers to 98 |
