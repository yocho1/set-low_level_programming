# C - Variables, if, else, while

This project introduces fundamental concepts in C programming including:
- Variables and data types
- Conditional statements (if, else if, else)
- Loops (while)
- Arithmetic and logical operators
- ASCII character set

## Files Description

| File Name | Description |
|-----------|-------------|
| `0-positive_or_negative.c` | Prints whether a random number is positive, negative, or zero |
| `1-last_digit.c` | Prints the last digit of a random number |
| `2-print_alphabet.c` | Prints the alphabet in lowercase |
| `3-print_alphabets.c` | Prints alphabet in lowercase then uppercase |
| `4-print_alphabet.c` | Prints alphabet except q and e |
| `5-print_numbers.c` | Prints single digit numbers |
| `6-print_numberz.c` | Prints single digit numbers using putchar |
| `7-print_tebahpla.c` | Prints alphabet in reverse |
| `8-print_base16.c` | Prints hexadecimal numbers |
| `9-print_comb.c` | Prints combinations of single digits |
| `10-print_comb2.c` | Prints combinations of two digits |
| `11-print_comb3.c` | Prints combinations of three digits |
| `12-print_comb4.c` | Prints combinations of four digits |
| `13-print_comb5.c` | Prints combinations of two two-digit numbers |

## Requirements

- All files compiled on Ubuntu 20.04 LTS with gcc using flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows Betty style
- No global variables
- Maximum 5 functions per file
